# Belief Examples

Three files. Each shows a different aspect of how belief works in practice.

| File | What It Shows |
|------|--------------|
| [belief-in-action.md](belief-in-action.md) | One document read by the belief system — what becomes a belief, what is filed silently, and why |
| [belief-over-time.md](belief-over-time.md) | One belief traced across six documents — how the statement sharpens and confidence builds |
| [fact-vs-belief.md](fact-vs-belief.md) | The same observation handled correctly (filed as a fact) and incorrectly (held as a belief) |

Start with `belief-in-action.md` to see the system working.
Read `belief-over-time.md` to understand how beliefs accumulate.
Read `fact-vs-belief.md` to understand the gate.
